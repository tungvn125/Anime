Anime CLI

For when you've finished Netflix and reality is just not cutting it.

Overview

Tired of endlessly scrolling for your next anime fix? Do you have more tabs open than you have friends? Let our Python-powered, AI-driven, coffee-brewing (okay, maybe not the coffee part... yet) toolkit do the heavy lifting.

It's like having a personal anime butler, but without the fancy uniform and the risk of them judging your questionable life choices.

Installation

Ready to join the elite? Of course you are. Just follow these sacred instructions. It's less painful than a filler arc, we promise.

Requires: Python 3.13, ani-cli

# Clone this repository
git clone https://github.com/tungvn125/Anime.git

# Go into the project folder
cd Anime

# Install some packages
pip install -r requirements.txt

Usage

Using this is easier than understanding the plot of Evangelion. Just tell the script what you want.

To see all the available commands, run:

python main.py --help

Or run without arguments to open the interactive Terminal UI (TUI):

python main.py

Interactive TUI

Run the program without arguments to open an interactive terminal UI inspired by gemini-cli. The TUI exposes shortcuts for Chat, Search, Recommend, Watch, Read LN, Genre Manager, Watchlist, and the new Manga Manager.

It uses simple-term-menu to provide a keyboard-friendly selection interface and submenus for related actions.

Manga Manager

The Manga Manager is a new feature available via the TUI and the AI chat assistant. It lets you:

  • Search for manga (interactive results with quick actions like view on MAL or add to readlist).
  • Get simple keyword-based manga recommendations.
  • Maintain a persistent readlist saved to readlist.json (add and list items).

Chat Function Calls

The AI chat assistant can call functions to perform actions like adding anime to your watchlist, opening an anime to watch, or managing manga readlist items. When the assistant requests a function call, the CLI executes it and returns the function result to the model; the assistant's follow-up message is shown immediately.
